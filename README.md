# Demographic Data Analyzer

A data analysis project, for FreeCodeCamp, that explores demographic information to uncover patterns related to education, income, work hours, and geographic distribution.

## Overview

This project analyzes census-like data to extract meaningful insights about:

- Income distribution  
- Education levels  
- Work habits  
- Demographic trends across countries  

The analysis is performed using Python and Pandas, focusing on real-world data exploration and statistical summaries.

## Features

### Data Analysis
- Counts of individuals by race  
- Average age of men  
- Percentage of individuals with a Bachelor's degree  
(Implemented in `demographic_data_analyzer.py`)

### Income Analysis
- Percentage of people with higher education earning >50K  
- Percentage of people without higher education earning >50K  
- Country with the highest percentage of high earners  

### Work Insights
- Minimum number of work hours per week  
- Percentage of high earners among minimum-hour workers  

### Country-Specific Insights
- Highest earning country and its percentage  
- Most common high-income occupation in India
